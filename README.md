# Treinamento de Versionamento Java

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de programação em Java juntamente com o uso do Git e GitHub para versionamento de código.

Durante o desenvolvimento foram aplicadas boas práticas de organização do projeto, utilização de commits para acompanhamento das alterações e gerenciamento do código-fonte utilizando controle de versão.

## 🎯 Objetivos

- Praticar programação em Java;
- Aprender e aplicar conceitos de versionamento com Git;
- Utilizar GitHub para armazenamento e gerenciamento do projeto;
- Desenvolver familiaridade com o fluxo de trabalho utilizado no desenvolvimento de software.

## 🛠️ Tecnologias Utilizadas

- Java
- Git
- GitHub
- Eclipse IDE

## 📂 Estrutura do Projeto

O projeto está organizado em pacotes e classes Java, seguindo uma estrutura simples para fins de aprendizado e treinamento.

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/Korion-Brito/Treinamento_de_vesionamento.Java.git
```

2. Abra o projeto em uma IDE Java (Eclipse, IntelliJ IDEA ou VS Code).

3. Execute a classe principal do projeto.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados:

- Controle de versão com Git;
- Criação e gerenciamento de repositórios no GitHub;
- Organização de projetos Java;
- Classes Abstratas;
- Estruturação de código orientado a objetos;
- Boas práticas de desenvolvimento.
